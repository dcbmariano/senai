# Back-end
Sistema de consulta de livros

## Requisitos
- PHP 8.1 ou superior
- Composer

## Instalação
- Renomeie o arquivo <code>env</code> para <code>.env</code>.
- A seguir, execute no terminal:<br>
<code>composer install</code>

## Testando => porta: 7777
No terminal, execute:<br>
<code>php spark serve --port 7777</code>
